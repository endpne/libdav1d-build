# libdav1d-build

build scripts for [libdav1d](https://github.com/haasn/libdav1d)

## Installation

### Swift Package Manager

```
https://github.com/mpvkit/libdav1d-build.git
```

## How to build

```bash
make build
# or specified platforms 
make build platforms=ios,macos
# or see help
make help
```
